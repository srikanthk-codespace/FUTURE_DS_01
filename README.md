📌 Overview

This project focuses on analyzing business sales data to identify key trends, top-performing products, category distribution, and regional performance.

The analysis helps in understanding business growth patterns and supports data-driven decision-making.

---

🎯 Objective

- Analyze sales trends over time
- Identify top-selling products
- Evaluate category-wise performance
- Understand regional sales distribution

---

🧠 Analysis Pipeline

Data Collection → Data Cleaning → Feature Creation → Data Analysis → Visualization → Insights

📊 Key Analysis Performed

📈 Key Insights

- Certain products contribute significantly to revenue
